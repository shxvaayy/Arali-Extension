# Extension Icons

Yahan extension ke icons rakhne hain:

## Required Icons:

1. **icon16.png** - 16x16 pixels (toolbar icon)
2. **icon48.png** - 48x48 pixels (extension management page)
3. **icon128.png** - 128x128 pixels (Chrome Web Store)

## Icon Design Tips:

- Simple aur clear design
- High contrast colors
- Recognizable symbol (camera/recording icon)
- PNG format with transparency

## Quick Icon Creation:

1. **Online Tools:**
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/
   - https://www.canva.com/

2. **Design:**
   - Camera/Video recording icon use karo
   - Background: Gradient (purple/blue)
   - Icon: White color
   - Size: 512x512 se start karo, phir resize karo

3. **Resize:**
   - Photoshop/GIMP use karo
   - Ya online tool: https://www.iloveimg.com/resize-image

## Temporary Icons:

Agar abhi icons nahi hain, to:
1. Koi bhi 16x16, 48x48, 128x128 PNG images rakh do
2. Baad mein proper icons replace kar dena
